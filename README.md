### 🏆 Featured Projects

#### 🔹 Intelligent Document Q&A Assistant (RAG)
A production-ready RAG application — upload PDF/DOCX/TXT and ask questions in plain English, getting **grounded, source-cited answers in under 1 second**. Retrieval with Sentence Transformers + FAISS, generation via Groq Llama-3.3-70B, PostgreSQL conversation memory, MLflow tracking, and a Streamlit UI — fully containerized with Docker Compose (4 services), **31/31 tests passing**.
`Python` · `FastAPI` · `FAISS` · `Sentence Transformers` · `Groq LLM` · `PostgreSQL` · `MLflow` · `Streamlit` · `Docker`

#### 🔹 AI Resume & Job-Match Assistant
An end-to-end assistant that matches job seekers to roles by **meaning** (Sentence-Transformer embeddings + cosine similarity over FAISS), parses resumes, runs **skill-gap analysis** for an **ATS-style match score**, and generates LLM-powered resume suggestions with a safe rule-based fallback. Service-oriented FastAPI backend with SQLAlchemy ORM, containerized with Docker.
`Python` · `FastAPI` · `FAISS` · `Sentence Transformers` · `LLM` · `SQLAlchemy` · `Docker`

#### 🔹 Customer Churn Prediction System

`Python` · `Scikit-Learn` · `FastAPI` · `MLflow` · `Docker`
